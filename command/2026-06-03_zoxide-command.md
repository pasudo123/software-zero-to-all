## zoxide

`zoxide` 는 자주 이동한 디렉터리를 기억해두고 짧은 키워드로 다시 이동하게 해주는 도구다.
단순히 경로를 외워서 `cd` 하는 대신, 사용 빈도와 최근 이동 기록을 기반으로 목적지를 찾아준다.

## 언제 쓰는가

- 자주 가는 프로젝트 디렉터리로 빠르게 이동하고 싶을 때
- 긴 경로를 매번 입력하기 번거로울 때
- 여러 후보 중 하나를 대화식으로 골라 이동하고 싶을 때

## shell 초기화

`zoxide` 는 shell 에 초기화 코드가 들어가 있어야 `z`, `zi` 명령을 편하게 쓸 수 있다.
zsh 를 사용한다면 보통 아래 형태를 shell 설정 파일에 둔다.

```bash
eval "$(zoxide init zsh)"
```

설정 후 새 터미널을 열거나 shell 설정 파일을 다시 로드한다.

## 기본 사용법

```bash
# coding 이 들어간 자주 가던 경로로 이동
z coding

# dotfiles 와 관련된 경로로 이동
z dotfiles

# 후보를 보고 선택해서 이동
zi
```

처음 설치한 직후에는 이동 기록이 없으므로 `z` 가 기대처럼 동작하지 않을 수 있다.
몇 번 `cd` 로 이동하거나 프로젝트를 오가면 점점 정확해진다.

## 여러 키워드로 좁히기

```bash
# coding 아래 software 관련 디렉터리로 이동
z coding software

# repo 이름 일부를 조합해서 이동
z zero all
```

키워드를 여러 개 넣으면 후보가 더 좁아진다.
정확한 전체 경로를 입력하지 않아도 되는 점이 `zoxide` 의 장점이다.

## zi 로 후보 선택하기

```bash
zi
```

`zi` 는 대화식 선택 모드다.
`fzf` 가 설치되어 있으면 후보 목록을 fuzzy search 로 고를 수 있어서, 비슷한 프로젝트가 많을 때 편하다.

## 상태 확인

```bash
# zoxide 에 저장된 디렉터리 기록 확인
zoxide query --list

# 특정 키워드에 매칭되는 경로 확인
zoxide query coding
```

## 주의사항

- `zoxide` 는 이동 기록을 기반으로 하므로 처음에는 학습 시간이 필요하다.
- shell 초기화가 빠져 있으면 `z` 명령이 없거나 기대와 다르게 동작할 수 있다.
- 비슷한 이름의 디렉터리가 많으면 `zi` 로 직접 고르는 편이 안전하다.
