## grep command 👀
* 입력으로 전달된 파일의 내용에서 특정 문자열을 찾고자할 때 사용하는 명령어이다.

## grep option
|grep option|desc|example|
|---|---|---|
|-r, --recursive|하위 디렉토리를 포함한 모든 파일에서 문자열 검색|grep -r {검색하고자 하는 문자열} {directory}|
